# Crime_analysis

This project focuses on analyzing a crime dataset from Kaggle to identify patterns in crime occurrences and understand how different demographic factors, such as gender and age, are affected.
The aim is to transform raw crime data into meaningful insights using data analysis and visualization techniques, helping to highlight critical areas for awareness and prevention.

- Objectives
  1.Identify the most common types of crimes
  2.Analyze gender-wise distribution of crime victims
  3.Study age group impact on crime occurrences
  4.Present insights using clear and impactful visualizations

- Work Done
  1.Cleaned and processed crime dataset
  2.Analyzed crime frequency across different categories
  3.Performed gender-wise analysis of crime impact
  4.Studied age group distribution, identifying vulnerable populations
  5.Created visualizations to highlight patterns and trends

- Key Insights
  1.Males are more affected than females across most crime categories
  2.Arson, Domestic Violence, Robbery, and Burglary are the most reported crimes
  3.Age group 71+ is significantly affected, indicating vulnerability in older populations
